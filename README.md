# Tower Defense Game

A Tower Defense game for teaching programming in Unity3d to my students.
It's made for a **Intermediate Level**. 

### Requirements
* You should know how to do basic programming.
* You should know the basics of the Unity Editor.

### Concepts 
* Transform
* Rigidbody
* Instantiate
* Damage enemies
